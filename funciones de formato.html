font funcion

2. Font

los fonts son herramientas fundamentales en el diseño gráfico y la comunicación visual. No solo embellecen el contenido, sino que también mejoran la legibilidad, comunican emociones y establecen una identidad de marca. 

	<"!DOCTYPE html>
<"html lang="es">
<"head>
    <"meta charset="UTF-8">
    <"meta name="viewport" content="width=device-width, initial-scale=1.0">
    <"title>Ejemplo de Fuente<"/title>
    
    <"link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
    <"style>
        body {
            font-family: 'Roboto', sans-serif;
        }
    <"/style>
<"/head>
<"body>
    <"h1>Hola, Mundo!<"/h1>
    <"p>Este es un ejemplo de texto usando la fuente Roboto.<"/p>
<"/body>
<"/html>

Los fonts permiten controlar la apariencia visual del texto en una página web, incluyendo el tipo de letra, el tamaño, el estilo (normal, cursiva, negrita), el color, etc.

form funcion

7.form

La etiqueta <"form> en HTML se utiliza para crear formularios interactivos que permiten a los usuarios enviar datos a un servidor. Esta etiqueta es fundamental para la recolección de información en aplicaciones web, como registros, inicios de sesión, encuestas y más.

		
		
			<"form action="/submit" method="POST">
    	<"label for="name">Nombre:<"/label>
    	<"input type="text" id="name" name="name" required>
    
    	<"label for="email">Correo Electrónico:<"/label>
    	<"input type="email" id="email" name="email" required>
    
    	<"label for="message">Mensaje:<"/label>
    	<"textarea id="message" name="message" required><"/textarea>
    
    	<"button type="submit">Enviar<"/button>
		<"/form>
