funciones de multimedia en html

video funcion

12. video

La etiqueta <"video> permite reproducir archivos de video directamente en el navegador, sin necesidad de plugins externos. Los usuarios pueden reproducir, pausar, detener y controlar el volumen del video, lo que proporciona una experiencia interactiva.

		
		
			<"head>
    <"meta charset="UTF-8">
    <"meta name="viewport" content="width=device-width, initial-scale=1.0">
    <"title>Ejemplo de Video<"/title>
<"/head>
<"body>
    <"h1>prueba de etiqueta de video<"/h1>
    <"p>A ejemplo:<"/p>
    
    <"video width="640" height="360" controls>
        <"source src="video.mp4" type="video/mp4">
        <"source src="video.ogg" type="video/ogg">
        Tu navegador no soporta la etiqueta de video.
    <"/video>
<"/body>

<"video width="640" height="360" controls>: Crea un elemento de video con un ancho de 640 píxeles y una altura de 360 píxeles. El atributo controls agrega los controles de usuario para reproducir, pausar y ajustar el volumen del video.

sonido funcion

13.sonido

La etiqueta <"audio> en HTML se utiliza para incrustar archivos de sonido en una página web. Al igual que la etiqueta <"video>, permite a los desarrolladores ofrecer contenido multimedia de forma efectiva, mejorando la experiencia del usuario.

		
		
			<"body>
    <"h1>Bienvenido a mi Página Web<"/h1>
    <"p>A continuación se muestra un audio de ejemplo:<"/p>
    
    <"audio controls>
        <"source src="audio.mp3" type="audio/mpeg">
        <"source src="audio.ogg" type="audio/ogg">
        Tu navegador no soporta la etiqueta de audio.
    <"/audio>
<"/body>

La etiqueta <"audio> permite reproducir archivos de audio directamente en el navegador, sin necesidad de plugins externos. Los usuarios pueden reproducir, pausar, detener y controlar el volumen del audio, lo que proporciona una experiencia interactiva.

object funcion

14.object

La etiqueta <"object> puede usarse para incrustar diferentes tipos de contenido, incluyendo imágenes, videos, y aplicaciones. Ofrece más control sobre cómo se presenta el contenido y permite especificar un fallback en caso de que el contenido no se pueda mostrar. Por ejemplo:

		
		
			<"object data="video.mp4" width="640" height="360">
 			 <"p>Tu navegador no soporta este formato de video.<"/p>
			<"/object>

Estas funciones permiten a los desarrolladores web enriquecer la experiencia del usuario al integrar una variedad de elementos multimedia en sus páginas.

tablas funcion

15.tablas

Las tablas en HTML se crean utilizando la etiqueta <"table>, que permite organizar datos en filas y columnas. Las tablas son útiles para presentar información estructurada de manera clara y accesible. A continuación, se describen las etiquetas principales que se utilizan para crear tablas:

		
		
			<"table>
  <"tr>
    <"th>Encabezado 1<"/th>
    <"th>Encabezado 2<"/th>
    <"th>Encabezado 3<"/th>
  <"/tr>
  <"tr>
    <"td>Dato 1<"/td>
    <"td>Dato 2<"/td>
    <"td>Dato 3<"/td>
  <"/tr>
  <"tr>
    <"td>Dato 4<"/td>
    <"td>Dato 5<"/td>
    <"td>Dato 6<"/td>
  <"/tr>
<"/table>
		
		
<2table>: Define el inicio de la tabla.

		
<"tr>: Define una fila de la tabla.

		
<2th>: Define una celda de encabezado, que generalmente está en negrita y centrada.

		
<"td>: Define una celda de datos.

marquee funcion

16.marquee

La etiqueta <"marquee> en HTML se utiliza para crear un efecto de desplazamiento horizontal o vertical de texto o imágenes dentro de una página web. Aunque esta etiqueta está obsoleta y no se recomienda su uso, aún se puede utilizar para lograr ciertos efectos visuales.

		
		
			
			<"marquee behavior="scroll" direction="left">
  			Texto que se desplazará
			<"/marquee>

behavior: Especifica el tipo de desplazamiento del texto. Los valores posibles son:

scroll: El texto se desplazará continuamente (predeterminado).

slide: El texto se desplazará una vez y luego se detendrá.

alternate: El texto se desplazará hacia adelante y hacia atrás.

style funcion

17. style

La función style en HTML se utiliza para aplicar estilos CSS (Cascading Style Sheets) directamente a un elemento HTML. Esto permite personalizar la apariencia y el diseño de los elementos de la página web.

		
		
			<"head>
 	 <"style>
   	 p {
    	  color: blue;
    	  font-size: 14px;
   	 }
 	 <"/style>
	<"/head>

button funcion

18. button

La etiqueta <"button> en HTML se utiliza para crear botones interactivos en una página web. Estos botones pueden realizar diversas acciones, como enviar formularios, ejecutar funciones JavaScript, o redirigir al usuario a otra página.

		
		
			<"button onclick="alert('Hola mundo')">Mostrar mensaje<"/button>
		

		
			<"form action="/submit" method="post">
  <"label for="nombre">Nombre:<"/label><"br>
  <"input type="text" id="nombre" name="nombre"><"br><"br>
  <"button type="submit">Enviar<"/button>
<"/form>

code funcion

19. code

La etiqueta <"code> en HTML se utiliza para representar un fragmento de código de computadora. Su función principal es semántica, ya que indica que el contenido dentro de esta etiqueta es un código fuente, lo que puede incluir lenguajes de programación, scripts, o cualquier otro tipo de código.

		
		
			<"p>Para imprimir "Hola Mundo" en JavaScript, puedes usar el siguiente código:

			<"code>console.log("Hola Mundo");<"/code>

div funcion

20. div

La etiqueta <"div> en HTML es un elemento contenedor que se utiliza para agrupar otros elementos HTML. Su función principal es estructurar y organizar el contenido de una página web, facilitando el diseño y la aplicación de estilos mediante CSS.

		
		
			<"div class="header">
    <"h1>Bienvenido a Mi Sitio Web<"/h1>
<"/div>

<"div class="content">
    <"p>Este es un párrafo de contenido principal.<"/p>
    <"img src="imagen.jpg" alt="Descripción de la imagen">
<"/div>

<"div class="footer">
    <"p>Derechos reservados © 2024<"/p>
<"/div>
