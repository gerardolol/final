imagen funcion

9.imagenes

La etiqueta <"img> en HTML se utiliza para insertar imágenes en una página web. Esta etiqueta es fundamental para enriquecer el contenido visual de un sitio, permitiendo a los desarrolladores y diseñadores web presentar información de manera más atractiva y efectiva. 

		
			<"img src="imagen.jpg" alt="Descripción de la imagen">

Este ejemplo ilustra cómo se puede utilizar la etiqueta <"img> para incluir una imagen en una página web, mejorando la experiencia del usuario y el atractivo visual del contenido.

fondos funcion

10.fondos

La función de establecer un fondo es proporcionar un color, imagen o patrón que se muestre detrás del contenido de la página. Esto puede ayudar a mejorar la estética visual y la experiencia del usuario.

		
			<"link rel="stylesheet" type="text/css" href="style.css">
			<"style>
			body{
			background-image: url(fondo.gif);
			background-position: center center;
			background-repeat: no-repeat;
			background-size: cover;
			background-attachment: fixed;
			<"/style>
	}

Para establecer un fondo en una página web, se suele utilizar la propiedad CSS background en combinación con una etiqueta HTML, generalmente el <"body>, aunque también se pueden aplicar fondos a otros elementos como <"div>, <"header>, etc.

desenfoque funcion

11. desenfoque

El efecto de desenfoque (blur) se utiliza para suavizar o difuminar los bordes y detalles de una imagen o elemento. Esto puede ser útil para crear un efecto de profundidad, enfocar la atención en el contenido principal o crear un efecto visual interesante.

		
		
		<"style>
        body {
            background-image: url('fondo.jpg');
            background-size: cover;
            background-position: center;
            filter: blur(5px); /* Aplica el efecto de desenfoque */
        }
        
        .content {
            padding: 20px;
            background-color: rgba(255, 255, 255, 0.7); /* Fondo semi-transparente para el contenido */
        }
   		 <"/style>

El efecto de desenfoque (blur) se utiliza para suavizar o difuminar los bordes y detalles de una imagen o elemento. Esto puede ser útil para crear un efecto de profundidad, enfocar la atención en el contenido principal o crear un efecto visual interesante.
